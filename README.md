# Minolta X-700 Service Manual

A community-digitized, web-readable service manual for the Minolta X-700 35mm SLR camera (models 2017-100 Chrome and 2017-200 Black).

**Live site:** `https://yourusername.github.io/x700-service-manual`

---

## Why this exists

The Minolta X-700 is one of the finest 35mm SLR cameras ever made, and thousands are still in active use. Original service manuals exist only as aging paper copies or low-quality scans scattered across the internet — difficult to read, impossible to search, and at risk of being lost entirely.

This project takes those scans, runs OCR correction, and presents the content as a clean, searchable, mobile-friendly web page that anyone can use for free.

## Parts index

The parts index is fully searchable and sortable. You can filter by:
- Part number (e.g. `2017-0401`)
- Page number
- Type (e.g. `screw`, `transistor`, `condenser`)

Part number flags:
- **※** — Exclusive to Chrome model (2017-100)
- **★** — Exclusive to both AE lock and non-AE lock variants
- **☆** — Exclusive to non-AE lock model only

## Contributing

If you have pages from this manual that aren't yet digitized, contributions are welcome.

**Easiest way:** Open an issue and attach your scan. Even a phone photo of a page is useful.

**If you want to contribute directly:**
1. Fork the repo
2. Add your scan to `/scans/` (name it by page number, e.g. `p21.jpg`)
3. Add the corrected text content to `index.html` in the appropriate section
4. Open a pull request

**What needs work most:**
- Exploded view diagrams (pages 1–19) — these need both the image embedded and the parts table alongside it
- Modification notes (P.21+) — critical reference for anyone using the ● marked parts
- Disassembly and adjustment procedures
- Circuit schematics (pages 18–19)

## Notes on accuracy

- OCR is corrected manually against the original scan
- Critical values (voltages, torque specs, shutter timing) should always be verified against the original scan before use
- Parts availability varies — cross-reference with current suppliers before ordering
- This manual covers the original 1983 production run; later revisions may differ
